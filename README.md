# GeoQuest — Interactive Prototype

A playable, real-map prototype of **GeoQuest** — a location-based knowledge-hunt game.

**Live demo:** see the GitHub Pages URL in this repo's settings.

Real map via MapLibre GL + CARTO basemaps; routing via OSRM. Your progress
(pearls, streak, vouchers, hunts) is saved in your browser. Everything is a
prototype — locations/rewards are simulated.
